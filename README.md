# Asfalto y Gas

Tienda online de equipamiento para motorista. Proyecto construido con React, Vite, Tailwind CSS v4 y Framer Motion.

## Características

- ✨ Diseño Premium y State-of-the-art
- 🛒 Checkout integrado con Redsys (Tarjeta y Bizum)
- 📦 Gestión de envíos con NACEX (Web Service — ver `api/NACEX_README.md`)
- 👤 Área de cliente personalizada
- 📱 Totalmente responsive y optimizado para móviles

## Desarrollo

```bash
npm install
npm run dev
```

## Seguridad

Este repositorio ha sido auditado y limpiado. Las variables de entorno sensibles se gestionan exclusivamente a través de los ajustes de despliegue en Vercel.
